Aeva UI is a modern design system library built with Lit web components for rapid web development. You can install it via npm: npm install aeva-ui This variant includes borders between items and an outer container border. Borders help distinguish sections more clearly in dense layouts. This accordion allows multiple items to be open at once. Both of these are open because of the multiple attribute. This item is interactive. You cannot see this content.
Property Type Description Default
label string The text displayed in the header. ''
open boolean Whether the item is expanded. false
disabled boolean Disables interaction with the item. false
Event Component Detail Description
toggle aeva-accordion-item { open } Dispatched when an accordion item is expanded or collapsed.
