Property Type Options Description Default
variant string text, password, email, multiline, integer, decimal The type of input and validation applied. text
appearance string bordered, flat Visual style of the input container. bordered
placeholder string - Hint text shown when input is empty. ''
rows number - Visible text lines for multiline variant. 3
min / max number - Numeric limits for integer or decimal. -
Event Detail Description
input { value } Dispatched when the input value changes (valid values only).
change { value } Dispatched when the input loses focus after a value change.
error { message, code, value } Dispatched when validation fails.
CSS Variable Description Default
--aeva-input-bg Background color of the input. -
--aeva-input-border-color Border color of the input. -
--aeva-input-border-radius Border radius of the input. 22px
--aeva-input-text-color Color of the entered text. -
--aeva-input-placeholder-color Color of the placeholder text. -
--aeva-input-focus-border-color Border color when the input is focused. -
