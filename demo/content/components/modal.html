Property Type Options Description Default
open boolean true, false Whether the modal is currently visible. false
backdrop boolean true, false Whether to show the overlay backdrop. true
close-on-backdrop boolean true, false Close when user clicks the backdrop. true
close-on-esc boolean true, false Close when user presses the Escape key. true
elevation number 1 - 5 Shadow depth level. 3
blur-amount number - Amount of backdrop blur in pixels. 10
Slot Description
default Main content of the modal. This can be structured HTML with title, body, and actions.
Method Parameters Description
close() - Closes the modal with a smooth fade-out animation.
Event Detail Description
open - Dispatched when the modal starts its opening animation.
close - Dispatched when the modal starts its closing animation.
backdrop-click - Dispatched when the user clicks on the backdrop.
CSS Variable Description Default
--aeva-modal-bg Background color of the modal content. rgba(255, 255, 255, 0.7)
--aeva-modal-backdrop-bg Background color of the backdrop overlay. rgba(0, 0, 0, 0.4)
--aeva-modal-border-radius Border radius of the modal container. 22px
--aeva-modal-padding Inner padding of the modal. 2rem
--aeva-modal-max-width Maximum width of the modal. 600px
