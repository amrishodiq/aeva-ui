Property Attribute Description Default
open open Whether the page is currently visible. false
background background Custom CSS background (color or gradient). 'white'
closeOnBackdrop close-on-backdrop Whether clicking the backdrop automatically closes the page. true
disableHistory disable-history If true, prevents the page from interacting with the browser history. false
