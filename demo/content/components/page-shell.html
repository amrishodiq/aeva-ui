Open Demo Page
Full Screen Page Close
This is a full-screen page component. It automatically handles slide animations and body scroll locking. 
<aeva-button onclick="page.show()">Open</aeva-button>

<aeva-page id="demo-page" background="..." close-on-backdrop>
    <div>Your Content</div>
</aeva-page>
Property Type Options Description Default
open boolean true, false Whether the page overlay is currently visible. false
background string CSS Color/Gradient Full-screen background style. 'white'
close-on-backdrop boolean true, false Automatically close when clicking empty space. true
disable-history boolean true, false Prevent creating browser history entry for this page. false
Event Detail Description
show - Fired when the page is opened.
close - Fired when the page is closed.
CSS Variable Description Default
--aeva-page-background Full-screen background style. white
--aeva-page-transition-duration Duration of slide animation. 0.4s
--aeva-page-z-index Stacking order of the overlay. 1000
