Property Type Description Default
variant string number, dots The visual style of the pagination. number
total number Total number of items. 0
current number The active page number (1-based). 1
page-size number Number of items per page. 10
sibling-count number Number of pages to show before and after current. 1
boundary-count number Number of pages to show at the start and end. 1
Event Detail Description
change { page: number } Fires when user clicks a page number or arrow.
