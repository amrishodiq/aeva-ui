Property Type Options Description Default
progress number 0-100 The current progress value. 0
mode string finite, infinite Type of progress animation. finite
shape string horizontal, circular Visual style of the indicator. horizontal
size string sm, md, lg Size of the progress indicator. md
label string - Descriptive text for the progress (horizontal only). ''
show-percentage boolean true, false Display numeric progress (circular only). false

This component currently does not dispatch custom events.
