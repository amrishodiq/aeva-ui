Try clicking or hovering over the stars!
Property Type Options Description Default
value number - The current rating value. 0
max number - Total number of stars. 5
interactive boolean true, false Allow users to change the value by clicking. false
allow-half boolean true, false Enables selection and display of half-stars. false
size string sm, md, lg Size of the star icons. md
show-label boolean true, false Display the numeric value label. false
Event Detail Description
rating-change { value } Dispatched when the rating value changes via interaction.

--aeva-rating-color Color of the filled stars. #fbbf24
--aeva-rating-empty-color Color of the empty stars. #e5e7eb
--aeva-rating-hover-color Color of stars during hover. #f59e0b
--aeva-rating-label-color Color of the numeric label. -
--aeva-rating-size-sm Star size for 'sm' variant. 16px
--aeva-rating-size-md Star size for 'md' variant. 24px
