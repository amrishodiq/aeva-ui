Property Type Description Default
header string Comma-separated list of column headers. Overridden if columns array is provided. ''
data string Data definition as a semicolon-separated list of comma-separated rows. Overridden if rows array is provided. ''
columns Array JS array definition for columns. Array of strings or objects like { key: '...', label: '...' } undefined
rows Array JS array definition for sequence of rows. Can be array of arrays or array of objects matching column keys. undefined
searchable boolean Enables a global search input above the table. false
searchPlaceholder string Placeholder text for the search input. 'Search...'
paginated boolean Enables slicing data and displaying a pagination control. false
itemsPerPage number Number of rows to display per page. 10
CSS Variable Description
--aeva-border-color Color of the table boundaries and cell separators.
--aeva-border-radius-md Corner radius of the table container.
--aeva-surface-color Background color of the table.
--aeva-surface-color-light Background color for the header row to differentiate it.
--aeva-text-color Text color for row cells.
--aeva-text-muted-color Text color for the header.
--aeva-hover-color Background color of a row upon being hovered.
