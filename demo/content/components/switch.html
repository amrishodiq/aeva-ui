Property Type Options Description Default
checked boolean - Whether the switch is active. false
disabled boolean - Whether the switch is non-interactive. false
label string - Text label to display next to the switch. ''
size string sm, md, lg The visual scale of the component. md
variant string default, glass The visual style applied. default
Event Detail Description
change { checked } Dispatched when the switch state is toggled.
CSS Variable Description Default
--aeva-switch-track-bg Background color of the track (unchecked). rgba(255, 255, 255, 0.1)
--aeva-switch-track-checked-bg Background color of the track (checked). var(--aeva-primary-color)
--aeva-switch-thumb-bg Color of the sliding thumb. #ffffff
--aeva-switch-transition Transition timing for state changes. 0.3s ease
