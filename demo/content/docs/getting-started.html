The fastest way to use Aeva UI is by importing the library from a CDN or using it in your modern build system.
Simply import the library and start using components:
import '@aeva/ui';

Framework Compatibility Built with Lit and standard Web Components, Aeva UI works seamlessly with any framework (React, Vue, Angular, Svelte) or no framework at all.
Prerequisites Aeva UI requires a modern browser environment. If using a build tool, Node.js 18+ is recommended.
Bundle Size Extremely lightweight. Each component is self-contained and tree-shakable, ensuring minimal impact on your application's load time.
Installation Install the package via npm, yarn, or pnpm:
npm install @aeva/ui
Setup Complexity Zero-config by default. Just import the components you need and they will automatically register themselves.
Code Snippet Standard usage in a TypeScript/JavaScript project: 
import '@aeva/ui/dist/components/atoms/aeva-button.js';
import '@aeva/ui/dist/components/atoms/aeva-text.js';

// That's it! Use them in your HTML:
// Welcome

Theming System Aeva UI uses a powerful CSS Variable-based theming system. You can switch between light and dark modes or create entirely new themes by overriding these variables at the root level.
How to Override Style To customize specific colors or spacings, simply define the corresponding CSS variables in your stylesheet: 
:root {
  --aeva-primary-color: #ff4757;
  --aeva-border-radius-md: 8px;
  --aeva-font-family: 'Inter', sans-serif;
}
