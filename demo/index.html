User Custom Styling

This is how user can customize the button's style.

                            
/* Global Customization */
aeva-button {
    --aeva-button-primary-bg: #34d399;
    --aeva-button-primary-hover-bg: #10b981;
}

/* Customization for a specific container */
.button-size aeva-button {
    --aeva-button-primary-bg: #f59e0b;
    --aeva-button-primary-hover-bg: #d68905;
    --aeva-button-border-radius: 8px;
}
