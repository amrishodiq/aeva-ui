Getting Selected Value

Two easy ways to get the selected value from a radio group

Get Value (Query) Get Value (Helper) Set to PayPal Check Selection
Click a button to see the result... 
                        
// Method 1: Direct Query (Simple & Fast)
const selected = document.querySelector('aeva-radio[name="payment"][checked]');
const value = selected ? selected.value : null;
console.log('Selected:', value);

// Method 2: RadioGroupHelper (Recommended)
import { RadioGroupHelper } from '@aeva/ui';

// Get selected value
const value = RadioGroupHelper.getValue('payment');

// Set selected value
RadioGroupHelper.setValue('payment', 'paypal');

// Check if any selected
if (!RadioGroupHelper.hasSelection('payment')) {
  alert('Please select a payment method');
}

// Get the radio element
const radio = RadioGroupHelper.getRadio('payment');

Usage Example

How to use the surface component in your code

                        <!-- HTML -->
<aeva-button id="openBtn">Open Modal</aeva-button>

<aeva-modal id="myModal" elevation="3" blur-amount="10">
  <h2>Modal Content</h2>
  <p>Your content here</p>
  <aeva-button id="closeBtn">Close</aeva-button>
</aeva-modal>

<!-- JavaScript -->
<script>
  const modal = document.getElementById('myModal');
  const openBtn = document.getElementById('openBtn');
  const closeBtn = document.getElementById('closeBtn');

  openBtn.addEventListener('click', () => {
    modal.open = true;
  });

  closeBtn.addEventListener('click', () => {
    modal.close();
  });

  // Listen to events
  modal.addEventListener('open', () => {
    console.log('Modal opened');
  });

  modal.addEventListener('close', () => {
    console.log('Modal closed');
  });
</script>

Animation Details

The page component features smooth animations:

  • Open: Slide in from right (100% → 0%), scale (120% → 100%), fade in (0 → 1)
  • Close: Slide out to right (0% → 100%), scale (100% → 120%), fade out (1 → 0)
  • Duration: 350ms with smooth easing
  • Back Button: Hardware/browser back button closes the page

Usage Example

                            
<!-- Basic Usage -->
<aeva-page id="myPage">
  <div style="padding: 2rem;">
    <h1>Page Title</h1>
    <p>Your content here</p>
  </div>
</aeva-page>

<!-- Custom Background -->
<aeva-page background="linear-gradient(135deg, #667eea 0%, #764ba2 100%)">
  <div style="padding: 2rem; color: white;">
    <h1>Gradient Background</h1>
  </div>
</aeva-page>

<!-- JavaScript Control -->
<script>
  const page = document.querySelector('#myPage');
  
  // Open page
  page.show();
  
  // Close page
  page.close();
  
  // Listen to events
  page.addEventListener('open', () => {
    console.log('Page opened');
  });
</script>
