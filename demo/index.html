CSS Customization

Customize typography using CSS variables

                                
/* Global Customization */
aeva-text {
    --aeva-text-font-family: 'Inter', sans-serif;
    --aeva-text-color: #2d3748;
    --aeva-text-link-color: #3182ce;
}

/* Customize specific variants */
aeva-text[variant="display"] {
    --aeva-text-display-weight: 900;
    --aeva-text-letter-spacing-display: -0.03em;
}

/* Customize states */
aeva-text[state="error"] {
    --aeva-text-error-color: #e53e3e;
}

/* Add spacing to headings */
aeva-text[variant^="h"] {
    --aeva-text-margin-bottom: 1rem;
}

/* Limit body text width for readability */
aeva-text[variant="body"] {
    --aeva-text-max-width: 65ch;
}

Input Variants

Six input types for different data collection needs

Password - Masked text input
Integer - Whole numbers only, no leading zeros, with min/max validation
Decimal - Numbers with decimals (e.g., 0.5, 123.45), with min/max validation
Email - Email address input with validation
Multiline - Textarea for longer text

Input Sizes

Three sizes to fit different contexts

Small (sm) - Compact input for tight spaces
Medium (md) - Default size for most use cases
Large (lg) - Prominent input for important fields

Regex Filtering (Text Variant Only)

Filter allowed characters using regex patterns - invalid characters won't be entered

Numbers Only - regex="^[0-9]*$"
Letters Only - regex="^[a-zA-Z]*$"
Letters and Spaces - regex="^[a-zA-Z ]*$"
Alphanumeric - regex="^[a-zA-Z0-9]*$"
Phone Number - regex="^[0-9\-\+\(\) ]*$"

Input States

Different states for user interaction

Normal - Default interactive state
Disabled - Non-interactive state
Required - Field must be filled

Validation & Error Handling

Built-in validation with error events - try entering invalid values!

Integer Validation - No leading zeros (01 ❌), min=1, max=100
Decimal Validation - Allows 0.5 ✓, but not 01.5 ❌, min=0, max=999.99
Text Max-Length - Cannot exceed 10 characters

    --aeva-input-focus-ring-color: rgba(52, 211, 153, 0.3);

/* Dark theme inputs */
.dark-theme aeva-input {
    --aeva-input-bg: #1f2937;
    --aeva-input-border-color: #4b5563;
    --aeva-input-text-color: #f9fafb;
}

/* Custom size */
aeva-input[size="lg"] {
    --aeva-input-padding-lg: 20px 24px;
    --aeva-input-font-size-lg: 20px;
}
