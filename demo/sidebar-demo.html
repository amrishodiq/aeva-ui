Responsive Sidebar Demo

This demo showcases the aeva-sidebar component with responsive behavior across different screen sizes.

Responsive Breakpoints

  • Mobile (<768px): Sidebar is hidden by default and appears as a drawer when toggled. Click the menu button in the top-left to open/close.
  • Tablet (768px-1024px): Sidebar is always visible in icon-only mode, showing just the icons without text labels.
  • Desktop (>1024px): Sidebar is always visible in full mode with both icons and text labels.

Try this: Resize your browser window to see the sidebar adapt automatically to different screen sizes!

Features

  • ✅ Automatic responsive behavior based on viewport width
  • ✅ Mobile drawer with backdrop overlay
  • ✅ Tablet icon-only mode for space efficiency
  • ✅ Desktop full mode with complete navigation
  • ✅ Smooth animations and transitions
  • ✅ Customizable width (sm, md, lg)
  • ✅ Glassmorphism effect option
  • ✅ Header and footer slots for branding and user info
  • ✅ Auto-close on navigation (mobile)
  • ✅ Accessible with proper ARIA attributes
